Three complete portfolio sites created (2026-03-14) in `/portfolio/` to demonstrate H&P's design capabilities to prospective clients.

1. **Embers & Oak (restaurant.html/css)** -- Upscale fine dining. Dark palette (near-black + copper + gold). Cormorant Garamond + Montserrat. Interactive tabbed menu, reservation form, scroll reveal animations.

2. **Groundwork Coffee (coffeeshop.html/css)** -- Warm neighborhood coffee shop. Cream + espresso brown + terracotta. Fraunces + DM Sans. Split hero layout, dot-leader menu, community events section, Instagram gallery placeholder.

3. **Root & Rise Studio (yogagym.html/css)** -- Yoga/fitness studio. Forest green + sage + coral accent. Playfair Display + Nunito Sans. Day-of-week schedule tabs, testimonial carousel with auto-advance, tiered pricing cards.

**Why:** These serve as real portfolio pieces hosted at hammerandpixels.com/portfolio/[name] to demonstrate design range to CT small businesses.

**How to apply:** Each site is self-contained (HTML + CSS, inline JS). All use image placeholders with descriptive labels for easy photo integration. All include "Website by Hammer & Pixels" footer credit linking to hammerandpixels.com. All are responsive with breakpoints at 480px, 768px, and 1024px.
