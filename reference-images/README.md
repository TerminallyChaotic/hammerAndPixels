# Reference Images for Hammer and Pixels

This folder contains free-to-use reference imagery for website design and content inspiration.

## Folder Structure

- **craftspeople/** — Artisans, makers, hands-on work
- **small-town/** — New England main streets, historic buildings, brick architecture
- **rivers-bridges/** — Water features, bridges, Farmington Valley landscapes
- **hipster-craft/** — Coffee shops, small businesses, artisan aesthetic

## Download Instructions

Use the search terms and platform recommendations in `REFERENCE_IMAGES.md` to:
1. Search for images matching your vibe
2. Verify the license (Unsplash/Pexels/Reshot = automatic green light)
3. Download and save to appropriate subfolder here
4. Organize with descriptive filenames

Example filename: `craftspeople/woodworker-workshop-natural-light.jpg`
