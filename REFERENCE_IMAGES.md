## How to Download Safely

### From Unsplash:
1. Go to unsplash.com
2. Search using terms above
3. Click the image you like
4. Click "Download Free"
5. License: Unsplash License (free for commercial use, no attribution required)

### From Reshot:
1. Go to reshot.com
2. Search using terms above
3. Create free account (email required)
4. Click image, then "Download"
5. License: Reshot Free (no attribution required)

### From Pexels:
1. Go to pexels.com
2. Search using terms above
3. Click image
4. Select size and download
5. License: Pexels License (free for commercial use, no attribution required)

### From Wikimedia Commons:
1. Go to commons.wikimedia.org
2. Search "Farmington Connecticut" or town names
3. Click image to view details
4. **CHECK LICENSE** (scroll down to see "License" section)
5. If it says "Public Domain" or "CC0" → Safe to use
6. If it says "CC BY" or "CC BY-SA" → Safe but **include attribution** (photographer name + "via Wikimedia Commons")
7. Click "Original file" to download high resolution
